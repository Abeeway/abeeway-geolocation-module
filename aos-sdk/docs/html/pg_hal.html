AOS HAL

Hardware Abstraction Layer

This section details the different AOS files that are part of the HAL.
Some of them, identified as HAL low layer files, are expected to be used only by the system but are exposed in the case where the user wishes to rewrite a driver or a facility on top of such files. Note that the ST HAL will not be described in this document. A lot of documentation can be found on the Internet.

HAL low layer

This software level resides between the ST HAL and the higher layers. It consists of:

  • aos_adc.h: Analog-to-Digital Converter driver.
  • aos-core/aos/aos_calib_hsi.h: HSI clock calibration. Internal use only.
  • aos_cc.h: Cycle Counter API, for precise execution time measurements using CPU cycles.
  • aos_cli_parser.h: CLI command parser. Internal use only.
  • aos_cli.h: Lowest layer of the Command Line Interface.
  • aos_common.h: Common AOS definitions.
  • aos_debug.h: Debug utilities.
  • aos_error.h: Hardware error handling.
  • aos_fifo.h: FIFO (byte oriented) utility.
  • aos_flash.h: Flash access.
  • aos_gnss_common.h: GNSS common definitions
  • aos_gnss_driver.h: Generic part of GNSS drivers.
  • aos_gpio.h: GPIO driver.
  • aos_hsem.h: Hardware semaphore driver.
  • aos_i2c_hal.h: I2C HAL.
  • aos_i2c.h: I2C driver.
  • aos_it_priority.h: Hardware interrupt/exception priorities.
  • aos_log.h: Log tracing facility.
  • aos_lpm.h: Low power driver.
  • aos_lr1110_gnss.h: LR1110 aided-GNSS driver.
  • aos_lr1110_mgr.h: LR1110 manager controlling the LR1110. LoRa driver.
  • aos_mcu_pins.h: MCU pin definition.
  • aos_pwm.h: PWM driver (used for buzzer).
  • aos_rf_switch.h: Radio Frequency switch manager.
  • aos_rtc.h: Real Time Clock driver
  • aos_spi_hal.h: SPI HAL.
  • aos_spi.h: SPI driver.
  • aos_system.h: System management.
  • aos_lpm.h: Low power manager.
  • aos_uart.h: LPUART/USART driver.
  • aos_wifi.h: WIFI driver (using the LR1110).
  • aos-sdk-version.h: AOS SDK version.
  • crc32.h: CRC32 calculation facility.
  • strnhex.h: String to hexadecimal facility.
