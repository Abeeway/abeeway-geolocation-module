◆ srv_geoloc_basic_action_t

enum srv_geoloc_basic_action_t

Action that can be done on a technology.

Note
It is recommended to have the always_acquire actions at the end of the scheduling list.
Enumerator
srv_geoloc_basic_action_none 

No action. Technology unused.

srv_geoloc_basic_action_always_acquire 

Always do the position acquisition regardless previous successes.

srv_geoloc_basic_action_skip_if_success 

If there is a previous success do not schedule this techno.

srv_geoloc_basic_action_count 

Number of actions.
