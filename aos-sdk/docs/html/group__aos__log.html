◆ aos_log_msg_feed()

void aos_log_msg_feed ( aos_log_module_id_t  mid,
aos_log_level_t  level,
const char *  format,
  ... 
)

Feed a log a message.

Parameters
midModule identifier
levelLog level of the message to be displayed.
formatString which accepts format qualifiers
Note
The function does not add the module prefix to the log. It should be used when the initial log message requires additional data to be displayed on the same line. The message is displayed only if the trace are enabled and if the module log level is greater than or equals to the level argument.

Example 1: 

 // Define a string
  static const char* _nice_str = "nice";
 // Start the log message
  aos_log_msg(aos_log_module_sys, aos_log_level_trace,"Hello %s world", _nice_str);
 // Terminate the message
  aos_log_msg_feed(aos_log_module_sys, aos_log_level_trace," . Smart log system.\n");

Will produce: 

 01:18:53.767272 (SYS) Hello nice world. Smart log system.

Example 2: Same example with aos_log_msg only. 

 // Define a string
 static const char* _nice_str = "nice";
 // Log the message
 aos_log_msg(aos_log_module_sys, aos_log_level_trace,"Hello %s world", _nice_str);
 // Log another message
 aos_log_msg(aos_log_module_sys, aos_log_level_trace," . Smart log system.\n");

Will produce: 

 01:18:53.767272 (SYS) Hello nice world. 01:18:53.767272 (SYS) Smart log system.
