Detailed Description

Pseudo range information for a given satellite.

Note
The unit of the pseudo-ranges depends on the hardware component, which provided them:
  • From the MT3333, the pseudo-ranges are expressed in centimeters if the compressed_prn (refer to the structure aos_gnss_satellite_prn_report_t) is set to false. Otherwise, it is expressed in steps of 0.059 nanoseconds modulo 2^24.
  • From the LR1110 (NAV1 mode), the pseudo-ranges are expressed in nanoseconds modulo 1 millisecond. In the case where the NAV2 mode has been selected, the PRN report is not provided,
