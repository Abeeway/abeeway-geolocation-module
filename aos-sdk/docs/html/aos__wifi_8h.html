Detailed Description

Basic LR1110 WIFI scan driver.

This file is inspired from the Semtech wifi_middleware. It interfaces AOS with the Radio planner. Note that the wifi_middleware is too restrictive in term of WIFI configuration and the number of reported Access points. That's the reason why it is overloaded by this driver,
