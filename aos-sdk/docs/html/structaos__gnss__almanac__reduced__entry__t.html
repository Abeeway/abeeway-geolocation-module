Detailed Description

Entry of a reduced almanac.

Note
This almanac entry is generated from the ephemeris. It is suitable for both the LR1110 and the MT3333. The data content contains either:
  • 19 bytes: almanac entry 15 bytes) plus the CA code generator (2 bytes) plus modulation bitmask (1 byte) plus constellation ID (1 byte).
  • 15 bytes: Almanac entry. Both reduced almanac can be converted to the standard almanac.
