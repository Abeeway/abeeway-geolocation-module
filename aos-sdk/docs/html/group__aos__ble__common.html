◆ aos_ble_std_services_callback_t

typedef void(* aos_ble_std_services_callback_t) (aos_ble_std_services_req_t req, void *arg, uint8_t *len)

BLE standard services callback. bridge between app layer and aos standard ble services.

Parameters
reqBLE standard services request type
argan argument that depend on the request, it could be used as input or output data and the type changes: aos_ble_std_srv_ias_alert: arg is an uint8_t input, for value description refer to https://www.bluetooth.com/specifications/specs/immediate-alert-service-1-0/ aos_ble_std_srv_lls_alert: arg is an uint8_t input, for value description refer to https://www.bluetooth.com/specifications/specs/link-loss-service-1-0/ aos_ble_std_srv_tps_read: arg is an int8_t output, for value description refer to https://www.bluetooth.com/specifications/specs/tx-power-service-1-0/ aos_ble_std_srv_bas_level_read: arg is an uint8_t output, for value description refer to https://www.bluetooth.com/specifications/specs/battery-service-1-0/ aos_ble_std_srv_bas_power_state_read: arg is an uint8_t output, for value description refer to https://www.bluetooth.com/specifications/specs/battery-service-1-0/ aos_ble_std_srv_ess_temperature_read: arg is an int16_t output, for value description refer to https://www.bluetooth.com/specifications/specs/environmental-sensing-service-1-0/
lenlength of arg in case of table or string
