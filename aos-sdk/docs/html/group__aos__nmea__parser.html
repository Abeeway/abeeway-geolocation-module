◆ nmea_extract_float_as_int()

int64_t nmea_extract_float_as_int ( char *  str,
uint8_t  nb_dec_digits 
)

Extract a float number and convert it as an integer.

Parameters
strInput string (null terminated)
nb_dec_digitsNumber of decimal digits we want to extract
Returns
the converted float
Remarks
The output value is multiplied by 10^nb_dec_digits. Example1: value 123.1, nb_dec_digits = 1, gives a value of 1231 Value is expressed in 1/10.

Example2: value 123.45, nb_dec_digits = 2, gives a value of 12345 Value is expressed in 1/100.

Example3: value 123.45, nb_dec_digits = 1, gives a value of 1234 Value is expressed in 1/10.

Example4: value 123.4, nb_dec_digits = 2, gives a value of 12340 Value is expressed in 1/100.

Example5: value 123, nb_dec_digits = 2, gives a value of 12300 Value is expressed in 1/100.
