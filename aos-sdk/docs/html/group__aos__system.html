◆ aos_gpio_bat_setup()

void aos_gpio_bat_setup ( aos_gpio_id_t  gpio_vbat_ctrl,
aos_gpio_id_t  gpio_vbat_analog,
uint8_t  vbat_ctrl_ratio,
uint16_t  vbat_ctrl_setup_delay,
int16_t  vbat_offset 
)

Initialize the battery monitoring setting.

Parameters
gpio_vbat_ctrlBattery monitoring. Voltage bridge control. If no voltage divider bridge, pass aos_gpio_id_last.
gpio_vbat_analogAnalog pin for which we want to read the battery level value.
vbat_ctrl_ratioVoltage bridge control ratio. E.g if the bridge divides by 2 then ctrl_ratio = 2. Value 0 is forbidden. If there is no bridge, the ctrl_ration equals 1.
vbat_ctrl_setup_delayTime in ms to stabilize the voltage divider bridge.
vbat_offsetvoltage offset introduced by the voltage divider bridge .
Note
The vbat parameters are used to read the battery level via the function aos_gpio_read_battery_level
