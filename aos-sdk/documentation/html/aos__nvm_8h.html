Detailed Description

Non-volatile memory interface API.

This module provides an interface to the non-volatile memory provided by AOS for application use.

Note
Write operations should be limited to the strict minimum to avoid deteriorating the underlying storage medium, notably FLASH memory which is only guaranteed for 10000 erase/write cycles.
