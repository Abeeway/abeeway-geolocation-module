Detailed Description

The cli_t structure should be private/opaque, really. But since we are doing plain C and don't have dynamic memory, that's just not possible - so we rely on the user not to use or modify anything in this structure.

Warning
This structure is NOT part of the API, its contents may change at any time without notice. You have been warned !
