Detailed Description

Cycle Counter API, for precise execution time measurements.

The cycle counter API is based on the Cortex-M DWT (Data Watchpoint and Trace), and can be used for precise timing measurements. Depending on the system clock speed, the cycle counter will wrap more or less quickly, so it should be used essentially for measuring reasonably short durations. Practical use is code optimization during development.

All functions are declared static inline to avoid the overhead of a function call.

Note
There is only one cycle counter in the system.
